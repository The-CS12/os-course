این ترد خیلی مهمه چون توش با مفاهیمی آشنا میشید که تا آخر این کورس نیازتون میشه.پس اگر آماده اید شروع کنیم.

### Processor (پردازنده):
کمتر کسی اسم پردازنده به گوشش نخورده.سخت افزاری که وظیفه پردازش دستور العمل ها,کنترل مسیر اجرای برنامه ها و اعمال تغییرات روی دیتای ها رو داره. تو اکثر مواقع منظور از پردازنده صراحتا **Central processing unit** یا *CPU* هستش!<br>

{{< readmore title="یکم بیشتر بدونید" open="false" >}}
کامپیوتر های امروزی با معماری `John von Neumann` (فون نویمان) ساخته میشن.این معماری پردازنده رو به دو بخش تقسیم میکنه: <br>
- ALU (Arithmetic,Logic Unit)
- CU (Control Unit)
{{< /readmore >}}

### Main Memory (حافظه اصلی):
تو این حافظه دیتای مربوط به برنامه ها و پروسه های در حال اجرا ذخیره میشه. حافظه اصلی **فرار** هستش یعنی بلافاصله بعد از قطع جریان برق (خاموش شدن سیستم) دیتای داخلش از بین میره. به حافطه اصلی `Real memory` یا ‍‍`Primary memory` هم میگن!

### I/O Modules (ماژول های ورودی و خروجی):
وظیفه این ماژول ها انتقال دیتا بین ماژول ها (میتونید برای راحتی ماژول ها رو دیوایس در نظر بگیرید) داخلی و خارجی هستش. مثل سنسور ها و یا دیوایس های ذخیره سازی.

### System Bus (باس ها):
باس ها مسیر های برای جا به جایی و انتقال دیتا بین پردازنده,مموری و I/O ماژول ها هستند.باس ها مثل **پستچی** دیتا ها رو بین بخش های مختلف جا به جا میکنن!
